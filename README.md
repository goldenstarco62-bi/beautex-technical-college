# Beautex Technical Training College Management System

A premium, fullstack management system for Beautex Elite Technical College.

## Features
- **Student Management**: Registration, ID generation, course tracking.
- **Faculty Management**: Staff profiles and department organization.
- **Modern UI**: Royal Maroon & Gold aesthetic with high-contrast inputs.
- **ID Card System**: Automatic generation of printable Student IDs with photo support.

## Project Structure
- `frontend/`: React + Vite application.
- `backend/`: Node.js + Express + SQLite API.

## Local Setup

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Deployment
- **Frontend**: Deploy `frontend` directory to Vercel.
- **Backend**: Use a persistent host (e.g., Railway, Render) for SQLite or migrate to a managed database (PostgreSQL/Supabase).
